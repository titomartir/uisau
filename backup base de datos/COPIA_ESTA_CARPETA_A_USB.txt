✅ ARCHIVOS LISTOS PARA COPIAR A USB:
═══════════════════════════════════════════════════════════════════

Este directorio contiene TODO LO QUE NECESITAS para hacer backup
y restaurar tu base de datos en otro equipo.

Archivos incluidos:
  ✓ database-manager.ps1 ← Usar este (menú interactivo)
  ✓ backup-database.ps1
  ✓ restore-database.ps1
  ✓ config.ps1 (configuración)
  ✓ LEEME.txt (guía principal)
  ✓ GUIARAPIDA.txt (resumen rápido)
  ✓ README-DATABASE-BACKUP.md (documentación completa)


📋 CÓMO USAR:
═══════════════════════════════════════════════════════════════════

EN TU EQUIPO ACTUAL (crear backup):
──────────────────────────────────
  1. Copia esta carpeta entera a tu USB
  2. Ejecuta: database-manager.ps1
  3. Selecciona opción 1 (Crear backup)
  4. El archivo .sql se guardará aquí
  5. Copia el .sql a la USB


EN EL SERVIDOR (restaurar):
───────────────────────────
  1. Copia esta carpeta de la USB a C:\ o escritorio
  2. Ejecuta: database-manager.ps1
  3. Selecciona opción 2 (Restaurar backup)
  4. Ingresa ruta del archivo .sql desde la carpeta


🚀 PASOS OPERACIONALES:
═══════════════════════════════════════════════════════════════════

PASO 1 - EN TU COMPUTADORA:
   ├─ database-manager.ps1 → Opción 1
   └─ Se crea: encuestas_satisfaccion-backup-YYYY-MM-DD-HHmmss.sql

PASO 2 - COPIAR A USB:
   ├─ Copia el .sql a: D:\backup base de datos\
   └─ (Junto con los demás archivos)

PASO 3 - EN EL SERVIDOR:
   ├─ Copia toda la carpeta de USB a: C:\
   ├─ database-manager.ps1 → Opción 2
   └─ Ingresa ruta: C:\backup base de datos\backup.sql

PASO 4 - LISTO:
   └─ npm start

🎯 RESUMEN:
═══════════════════════════════════════════════════════════════════

Cuando necesites:
  1. Lleva USB a tu computadora → Ejecuta database-manager.ps1
  2. Crea backup → Copia .sql a USB
